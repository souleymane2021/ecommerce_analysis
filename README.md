# Projet SQL - Analyse des ventes d'une boutique en ligne

Ce projet contient le code SQL complet pour analyser les ventes d'une boutique en ligne :
- Création des tables (clients, produits, ventes)
- Insertion de données fictives
- Requêtes d'analyse : ventes par catégorie, par mois, top clients, indicateurs globaux
- Utilisation de PostgreSQL

Le code SQL complet est dans le fichier `ecommerce_analysis.sql`.
